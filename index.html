Poké Pocket Promos API

An API for promo cards in Pokémon TCG Pocket

Pokémon Resource

Contains a Pokémon's basic stats, their attacks, the set in which it was released, the artist, and the link to its image which is hosted on this site!

Item Resource

Contains an item's basic stats, its text, the set in which it was released, the artist, and the link to its image which is hosted on this site!

Supporter Resource

Contains an supporter's basic stats, its text, the set in which it was released, the artist, and the link to its image which is hosted on this site!

Release Resource

Contains a Releases' release date, the total number of cards, a description about the set, and a link to a YouTube video to embed!
